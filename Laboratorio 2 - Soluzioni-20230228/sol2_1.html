Soluzione 9.1

Il metodo iterativeBinSearch effettua un accesso all'elemento intermedio dell'array ordinato, e se questo non contiene il valore cercato itera la ricerca sul semi-array superiore o inferiore, fino a quando
  1. non trova il valore cercato, oppure
  2. l'intervallo su cui effettuare la ricerca non diventa vuoto
Prestare attenzione al fatto che il confronto di ordinamento tra stringhe avviene attraverso i metodi equals e compareTo. 
public class StringArrayAlgs

    public static int iterativeBinSearch(String[] v, int vSize, String s)
    {
        int from = 0;
        int to = vSize - 1;
        while (from <= to)
        {
            int mid = (from + to) / 2;
            if (s.equals(v[mid]))
                return mid;
            else if (s.compareTo(v[mid]) < 0)
                to = mid - 1;
            else
                from = mid + 1;
        }
        return -1;
    }
