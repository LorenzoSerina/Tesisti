Soluzione 9.2

Il metodo bubbleSort, descritto nel testo dell'esercizio, necessita di due cicli annidati. Inoltre abbiamo isolato l'operazione di scambio tra due elementi dell'array in un metodo ausiliario swap.
Prestare attenzione al fatto che il confronto di ordinamento tra stringhe avviene attraverso il metodo compareTo. 
public class StringArrayAlgs
{

   //... da completare

    public static void bubbleSort(String[] v, int vSize)
    {
        for (int i = 0; i < vSize-1; i++)     //attenzione alla formulazione 
            for (int j = vSize-1; j > i; j--) //di questi due cicli
                if (v[j].compareTo(v[j-1]) < 0)
                    swap(v, j, j-1);
    }
    private static void swap(String[] v, int i, int j)
    {
        String s = v[i];
        v[i] = v[j];
        v[j] = s;
    }

}
